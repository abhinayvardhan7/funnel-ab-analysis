# funnel-ab-analysis

Funnel & A/B Testing Analysis
Problem

Users were visiting the platform and exploring products, but many of them were not completing purchases.
The goal of this analysis is to understand where users drop off in the funnel and how to improve conversion.

What I Did
Performed funnel analysis to track user journey
Identified drop-off points
Conducted A/B testing to compare two versions
Funnel Steps

Visit → View Product → Add to Cart → Purchase

Results
Visitors: 41
Viewers: 35
Cart Users: 26
Purchasers: 17
Conversion & Drop-off
Visit → View: 85% (15% drop-off)
View → Cart: 74% (26% drop-off)
Cart → Purchase: 65% (35% drop-off)
Insights

Most users drop at the checkout stage (after adding items to cart).

This suggests that:

the checkout process may be confusing
payment options may be limited or difficult
users may face unexpected costs
A/B Testing Results
Group A: 26.6% conversion
Group B: 50.0% conversion

Group B performs significantly better than Group A, indicating that the new version improves conversion.

Final Understanding

Funnel analysis helped identify the problem (drop-off at checkout).
A/B testing helped identify a better-performing solution (version B).

Recommendation
Improve checkout flow by simplifying steps
Add better and more flexible payment options
Implement version B to increase overall conversion
Tools Used
